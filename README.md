# yogiyocbnu



## 근처가게들을 gps를 기반으로한 지도 검색으로 그가게의 전화번호와 웹사이트 접근 가능    

## 지도는 포켓몬 고 라는 유명한 앱에 사용된 openstreetmap 사용 이에따라 사용자들이 지도에 기여할수있다는걸 표현 

## 전주는 아직 활성화가 되어있지않아서 맛집분석은 윤서가 만들은 웹 (다음지도)사용
